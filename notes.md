# TypeScript

## How To install TypeScript :-
```bash
npm i typescript -g
```
## How To Check version of TypeScript:-
```bash
tsc -v

```
## How To runTypeScript File :-
- This command will create a new js file with same name of TypeScript file.
```bash
tsc fileName.ts

#  in watch mode
tsc -w
```

## How To Create TypeScript Config File :-

```bash
tsc --init
```
